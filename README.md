# AI Personal Assistant-n8n

## Overview

AI-powered personal assistant built using n8n and Groq LLM. The assistant can manage calendar events, track expenses, automate emails, and perform web searches through natural language commands.

## Features

* Google Calendar Integration
* Expense Tracking with Google Sheets
* Gmail Automation
* Web Search with Tavily
* AI Agent Powered by Groq
* Webhook API Support

## Tech Stack

* n8n
* Groq LLM
* Google Calendar API
* Google Sheets API
* Gmail API
* Tavily Search API

## Workflow Architecture

Webhook → AI Agent → Tools → Response

## Setup

1. Import `workflow.json` into n8n
2. Configure API credentials
3. Activate the workflow
4. Use the production webhook URL
